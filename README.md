# 蔡月明课题组内部平台系统（独立页面版）

## 系统概述
这是一个专为蔡月明课题组设计的内部管理平台，采用独立页面设计，每个课题有单独的详情页面。

## 当前文件结构
- index.html - 课题总览页面
- topic1.html - 痛风与高尿酸血症课题详情
- topic2.html - 类风湿关节炎课题详情
- topic3.html - 单细胞多组学课题详情
- topic4.html - 天然免疫与炎症信号通路课题详情
- topic5.html - 临床转化与药物靶点课题详情
- topics_data.json - 课题数据文件
- update_topics.py - 课题信息更新脚本
- PROJECT_STRUCTURE.md - 项目结构说明
- 1.txt - 项目需求文档
- README.md - 项目说明文档

## 功能特点
1. 主页显示所有课题列表，点击可跳转到对应详情页
2. 每个课题有独立的详情页面
3. 每个详情页只显示该课题的完整信息
4. 页面间可快速跳转
5. 静态页面，易于部署

## 部署说明
1. 将所有HTML文件上传到Web服务器或托管平台
2. 确保所有文件在同一目录下以保证页面间正确跳转
3. 访问入口为index.html

## 使用指南
- 通过index.html查看所有课题列表
- 点击课题名称跳转到对应的独立详情页面
- 在详情页可通过"返回课题总览"回到首页

## 维护指南
如需更新课题信息，可以使用以下方法：
1. 直接编辑对应的topicX.html文件
2. 修改topics_data.json文件，然后运行`python update_topics.py`来更新所有HTML文件
3. 运行`python update_topics.py`，使用交互式界面更新信息

运行更新脚本的步骤：
1. 打开命令行，进入项目目录
2. 运行 `python update_topics.py`
3. 按照提示操作
